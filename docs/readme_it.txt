PREPARAZIONE
===========

Per installare thirty bees, hai bisogno di un server remoto o nel tuo computer (MAMP), con accesso ad un database come MySQL.
Dovrai accedere a phpMyAdmin per creare un database e per indicare le informazioni nel database all'installatore.

Sei non hai un host e non sai come creare il tuo negozio, ti offriamo un negozio già pronto che ti farà realizzare il tuo negozio online in meno di 10 minuti senza nessun bisogno di competenze tecniche.
Ti invitiamo a visitare:
	https://thirtybees.com/blog/thirty-bees-on-cloudways/

INSTALLAZIONE
============

Devi solo andare nella tua directory thirty bees e usare l'installatore :-)

Se si verifica un qualsiasi errore PHP, forse non hai PHP5 oppure devi attivarlo nel tuo web host.
Vai nel nostro forum e cerca le impostazioni di pre-installazione (PHP 5, htaccess) per alcuni hosting services (1&1, Free, Lycos, OVH, Infomaniak, Amen, GoDaddy, etc).

Impostazioni specifiche per web host in inglese:
    https://docs.thirtybees.com/installation/#server-requirements

Se non trovi nessuna soluzione per lanciare l'installatore, scrivi un post sul nostro forum:
	https://forum.thirtybees.com/category/5/installation

Troverai sempre una soluzione ai tuoi problemi :-)
